Пазлы

Пазл представляет собой картинку, разделенную на множество мелких деталей разной формы. Цель игры - собрать их всех воедино.

Начать игру можно несколькими способами. Но в первую очередь необходимо установить уровень сложности. Для этого выбрать одно из предложенных разбиений (3x3, ..., 9x9) и по желанию поворот деталей. По умолчанию устанавливается самый простой уровень: 3x3 без поворота.

После выбора опции "Случайная картинка" приложение самостоятельно выберет случайную категорию и случайную картинку категории из имеющихся.
Вы можете кликнуть по одному из изображений категорий, тогда раскроется список всех картинок, принадлежащих категории. Аналогично, выбрав опцию "Случайная картинка", будет выбрана любая картинка данной категории

После того, как игра начнется, нужно перетаскивать детали пазла по полю, оставляя их в нужных ячейках. В случае выделения детали, она начинаем подсвечивается красным цветом. Если установлен поворот детали, то двойным щелчком можно его повернуть на 90°.
